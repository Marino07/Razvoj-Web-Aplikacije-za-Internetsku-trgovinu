
## About App
This is a complex e-commerce application built with Laravel. It allows users to browse products, manage their shopping cart, and complete purchases using Stripe for secure payment processing.
## Technologies Used
- Laravel
- MySQL
- Stripe API
- Bootstrap
- SCSS
- Vanilla JS
- Apache
